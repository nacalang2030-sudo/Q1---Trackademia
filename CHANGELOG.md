Added assignment tracking system.

Added class schedule feature.
Implemented Add Schedule and View Schedule features.

Version 1.4 – February 10, 2026

Improved input validation handling.
Cleaned repeated header print statements. 
Enhanced overall interface consistency.

Version 1.5 – February 15, 2026

Fixed invalid choice message bug (missing quotation mark). 
Final debugging and minor formatting improvements.
General performance polishing.

Version 1.6 – March 4, 2026

Added a login system for authentication. 
Improved security and user access control.

Version 1.7 – March 12, 2026

Converted program interface to Tkinter GUI.
Enhanced graphical user interface and overall user experience.

Version 1.8 – April 3, 2026

Refactored system into a fully GUI-driven tabbed interface using ttk.Notebook.
Removed text-based menu loop and replaced with event-driven buttons.
Simplified navigation between Grades and Assignments modules.

Version 1.9 – April 9, 2026

Implemented JSON-based data persistence.
Added Save Data and Load Data features. Introduced file validation, error handling for missing or corrupted files, and user feedback via message boxes.

Version 2.0 – April 15, 2026

Enhanced Grades system by adding unit-based grading.
Implemented weighted GWA computation instead of simple averaging.
Improved grade display formatting with tabulated output. 
Added delete and clear-all functions.

Version 2.1 – April 18, 2026

Upgraded Assignment system with deadline tracking. 
Added automatic calculation of days remaining using datetime. 
Introduced deadline status indicators (Overdue, Due Today, Due Tomorrow, Soon).
Added mark-as-done functionality and status labels (Done/Pending).

Version 2.2 – April 21, 2026

Added dashboard information bar displaying real-time GWA and number of pending assignments.
Improved UI styling with custom colors, hover effects, and consistent layout. Enhanced input validation and user feedback across all features.
